# VestalFocuser STM32 + A4988

STM32F103C6T6 + A4988 步进电机驱动版本。
Vestaline 协议 100% 兼容 ASCOM/INDI 驱动。

## 硬件连接

```
STM32F103C6T6 (Blue Pill)        A4988 Driver
─────────────────────────        ────────────
PB0  ────────────────────────→   STEP
PB1  ────────────────────────→   DIR
PB10 ────────────────────────→   ENABLE
PB12 ────────────────────────→   MS1
PB13 ────────────────────────→   MS2
PB14 ────────────────────────→   MS3

    GND ────────────────────→   GND
    GND ────────────────────→   SLP (接 GND 避免休眠)
    GND ────────────────────→   RST (接 GND 避免复位)

A4988                          NEMA17 步进电机
──────                          ────────────
2B  ────────────────────────→   B-
2A  ────────────────────────→   B+
1A  ────────────────────────→   A+
1B  ────────────────────────→   A-

VMOT ────────────────────────→   8-35V 电源 +
GND  ────────────────────────→   电源 GND

VDD  ────────────────────────→   3.3V (STM32)

NTC 热敏电阻:
   3.3V ── 10kΩ ──┬── PA0 ── NTC ── GND

调试串口 (可选):
   PA2 ── USB-TTL RX (115200 bps)
```

## 与 A328 版本对比

| 特性 | A328 (Arduino Nano) | STM32 (Blue Pill) |
|------|---------------------|-------------------|
| MCU | ATmega328P 16MHz | STM32F103C6T6 72MHz |
| 驱动 | ULN2003 (4 相) | A4988 (STEP/DIR) |
| 微步 | 软件半步 (8 拍) | 硬件 16 微步 (MS pins) |
| 步/转 | 4096 (软件) | 4096 (200×16, 硬件) |
| ADC | 10-bit | 12-bit |
| EEPROM | 有 (1024B) | 无 |
| 速度上限 | ~666 Hz | ~5000 Hz |
| RAM | 2 KB | 10 KB |
| Flash | 32 KB | 32 KB |

## 编译

需要 Arduino_STM32 开发环境:

1. 安装 Arduino IDE
2. 添加 STM32 开发板: https://github.com/stm32duino/Arduino_Core_STM32
3. 选择 Board: Generic STM32F103C6
4. Upload method: STLink / Serial / STM32CubeProgrammer

## 微步 vs 半步

| 项目 | ULN2003 半步 | A4988 16微步 |
|------|-------------|-------------|
| 原理 | 软件切换励磁相位 | 硬件 DAC 细分电流 |
| 精度 | 8 细分/4 物理步 | 16 细分/物理步 |
| 噪音 | 较大 (共振明显) | 低 (正弦波驱动) |
| 扭矩 | 低 (半步=50%扭矩) | 较高 (正弦峰值保持) |
| 丢步率 | 较高 | 极低 |

## 注意事项

1. **无 EEPROM**: STM32F103C6T6 断电丢失位置，每次上电从 0 开始
2. **MS 引脚**: 已在固件中拉高 (16 微步)，不需要外部跳线
3. **ENABLE 引脚**: 空闲时拉高 (A4988 休眠)，移动时拉低使能
4. **VMOT 电压**: 8-35V，推荐 12V，需加 100μF 电解电容滤波
5. **逻辑电压**: STM32 3.3V 可直接接 A4988 的 5V 容忍引脚
6. **ASCOM MaxStep**: 建议从 16384 开始，可根据实际行程调大
